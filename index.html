ТЕОРИЯ ГРАФОВ

Что такое граф?

Определение

Граф — это набор вершин и рёбер.

G = (V, E)

  • V — вершины (узлы)
  • E — рёбра (связи)

Типы графов

Ориентированный

Рёбра имеют направление

A → B → C

Неориентированный

Рёбра без направления

A — B — C

Взвешенный

Рёбра имеют вес

A—[5]—B—[3]—C

Связный

Все вершины связаны

все соединены

Где используются графы?

Соцсети

Друзья = вершины
Дружба = рёбра

Навигация

Города = вершины
Дороги = рёбра

Сети

Компьютеры = вершины
Соединения = рёбра

Биология

Гены = вершины
Взаимодействия = рёбра

7 МОСТОВ

Задача о 7 мостах

Формулировка

Можно ли пройти по всем 7 мостам Кёнигсберга, проходя по каждому только один раз?

Графовая модель

  • 4 участка земли = 4 вершины
  • 7 мостов = 7 рёбер

Решение задачи

Теорема Эйлера

Эйлеров цикл существует ⇔ все вершины имеют чётную степень

Для Кёнигсберга:

АЛГОРИТМ ДЕЙКСТРЫ

Алгоритм Дейкстры

Что делает?

Находит кратчайший путь во взвешенном графе

Особенности

  • Только неотрицательные веса
  • Жадный алгоритм
  • Сложность: O(V²)

Как работает?

1

Инициализация

Начальной вершине: 0
Остальным: ∞

2

Выбор вершины

Берём вершину с минимальным расстоянием

3

Обновление

Пересчитываем расстояния до соседей

4

Повторение

Повторяем шаги 2-3 для всех вершин

Пример графа

A (0) ---[6]---> B (∞)
A (0) ---[2]---> C (∞)

ПРАКТИКА: КАРТА

Ортодромы

Что это?

Кратчайший путь между двумя точками на сфере

Формула

d = R × arccos(sin φ₁ sin φ₂ + cos φ₁ cos φ₂ cos Δλ)
R = 6371 км (радиус Земли)

Калькулятор расстояний
